CSS can be added to HTML documents in 3 ways:

  • Inline- by using the style attrinbute inside HTML elements
  • Internel- by using a style element in the head section
  • External - by using a link element to link to an external css file.
  • The most common way to add css, is to keep the style in external css files. However In this tutorial we will use inline and internal styles.because this is easier to demonstrate. and easier for you try it yourself.

When compare to the Css class Vs ID,the difference is that css class applies a style to multiple elements.ID, on the other hand, applies a style to a one unique element. ID is also a special in that you can use the special URL to link directly to an element and its used by Javascript.

In case , selectors are used to target a specific or range of elements on a web page . once an elements has been targeted, a style or set can be apllied to the elements
